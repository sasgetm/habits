# Habit Tracker

Минималистичный трекер привычек на Vue 3. Приложение позволяет отслеживать выполнение ежедневных привычек через простой и наглядный интерфейс. Для хранения данных используется Google Sheets.

## Текущий функционал

- Отображение списка привычек
- Отметка выполнения с несколькими уровнями
- Добавление / редактирование / удаление привычек
- Просмотр истории выполнения привычек
- Сохранение временных данных в localStorage
- Синхронизация с Google Sheets

## Технологии

- Vue 3
- Vite

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение будет доступно в режиме разработки.

## Сборка проекта

```bash
npm run build
```

## Предпросмотр сборки

```bash
npm run preview
```
